# **第一份工作**

## **写在前面**

之前呢，有在一高端民宿兼职过两个暑假，很棒很舒服的一段经历，但算不上一份正式的工作。

借着大学实习的时间想着最后一年要回学校的次数也屈指可数，直接去想去的城市玩吧，顺便直接找份正式工作好活着，毕竟家里是不会再给一分钱的了！

6月底，咱来到昆明，浅跑了N家公司后最后入职现公司。其实还没了解时并不是很满意，既不包吃也不包住，工资也一般，不过想想咱一个人在昆明也足够，其他大部分时候让人很满意，也很有趣，就安顿下来了。

## **有害生物防治行业**

先说一下咱是干啥的再说一些实际情况吧！

我被分配在政府组，简而言之就是合同的对象是政府(云南的各个县市镇都有分布)；当然公司还有商业.机场.铁路组，有时候我们会去出差帮忙。我们的工作主要分为两类:四害(鼠.蚊.蝇.蟑)的防治和（监测与评估）。

=== "四害防治"

    - 去毒饵站放鼠药；处理积水抑制蚊蝇幼虫繁殖；使用超低容量喷雾，热烟雾机，烟雾弹对蚊蝇蟑成虫进行直接消杀。

=== "四害检测与评估"

    - 去某个区域(如昆明西山区)做四害的密度监测和侵入率调查。会去一定量的餐馆.超市.小区.医院.公园.村庄.机关单位.企业.大小型水体去观测去取样。

说的简单，但是要真正按规定做是很麻烦的，但是这是政府招标的嘛，质量就可想而知了，还要求我们密度监测出来必须小于某个数值（笑。所以政府组一直是公司最水的组。

## **工作记录**

- 先说明一下我们的工作证明是施工照片和施工单，没照片你干再多也没有用。所以有些组长就很水，专业拍照片忽悠各级（笑

在昆明的话是朝8晚5，8.半多我们到公司等各个小组人到齐后就开车出发向工作地点，消杀啥的是在街道社区的带领下去老旧小区等地，所有一般9.半我们到后他们也上班了开始工作。如上所说，活是不累的，就是一直在走啊走，找地方投药打药。平均情况上午两小时下午两小时做四个社区差不多，特殊情况也常见，有些社区老旧小区很少，我们也是做3-4个，可能一个上午就做完了，然后吃了午饭去歇息了；昆明所有的区也基本是逛遍了，整个昆明的监测评估防治咱们公司都有在做。

咱们也会经常出差，咱挺喜欢的，包吃包住，住的一般，普通酒店，吃的就很棒了，天天就是下馆子，火锅，烧烤，还能尝到各地不同的美味。目前去了：嵩明县，石林彝族自治县，碌劝彝族苗族自治县，澜沧拉祜族自治县，镇康县。注意这个镇康，是直接与缅北接壤的一个县哦，咱们开了一天车才到，很多地方我们工作的时候对面就是缅甸的土地，很有趣。

短时间内这是一份很棒的工作，基本没有加班，上班时间比较轻松有趣，对于我这种喜欢到处跑在外面闲逛的来说简直福音；工资是有点低，但每天就上那么几小时的班，还跟玩一样的，在昆明的这两年就先这样吧，正好有很多时间可以学更多的东西。
