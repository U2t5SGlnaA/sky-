

# **初见! -- 广东**

本无期待，但却收获满满！  
来年再见！

<!-- uptoc -->

### **初见**

中秋前一周，想着去哪里玩一下，鉴于昆明已经被我跑遍了(益得于工作原因<(￣︶￣)↗)，各地州我想骑行去，所以就没有在云南考虑了，遂看了看一些航班，发现中秋去广州的航班价格挺不错，一点就买了，也没做啥计划，应该是根本没有计划(￣▽￣)"

## **主线**

凌晨3时，抵达广东上方，空中月球闪亮，城市灿若繁星。又几分后，落于白云国际机场，这次的故事从这里开始：

下舱门，来自空气的问候：燥热，嗯，是广东啊！Sky在昆明居住多月，感到丝丝不适，急忙上车前往航站楼。

距首班地铁开:3小时。在机场浅走后，开启音乐，找了位置开始浅浅的休息。很快便6时，正式启程出发广州南站，第一班车，旅客已很多，磕磕碰碰，抵达广州南。

出站，欸！嗯？周围啥也没有，再转转吧，欸欸欸，还是啥都没有。转移阵地，打车前往了一个叫美的大道的地方。旁边的河道里好多垃圾，里面的鱼看起来比我还热，一只只都漂浮于水上呼吸。不是好景，看了地图去美的楼那边喝了咖啡。

教父冷萃，好高级，也有些贵，只是味道就强差人意了，好吧，来周边转转，发现了是北滘公园，还有一个24小时图书馆

图书馆人好多，大多数是学习的呢，还有讨论题目，所以有点吵，在此休整后去公园逛了逛，有中秋活动和相亲大会。可惜都不符合我的胃口。

一份公交之旅开始啦：从北滘公园出发坐K335前往天佑城，耗时半个时辰多，中途于保利百合花园下车，换乘328路前往终点站兴涛苑。总方向是顺德。

很幸运，公交车上都并无多少人，我可以找一个后排靠窗的位置欣赏一路的风景，听一路上的人讨论各种事情，(虽然大部分听不懂.笑)，车上其实是很颠颠坡坡，摇摇晃晃的，大多数人应该不会喜欢以这种方式来游览吧。

一路上从CBD到工业园区，中间还有一些村庄，还有好多的工厂，路上的风景其实并无太大的惊艳，但是是从未见到过的景色，不知道会在哪一处就让心里惊动了一下呢！

到兴涛苑这里，步行过两条马路，踏入泥泞路，穿过菜园和人工林子，可以看到容桂水道了，边上还有河中的洲，青草旺盛。岸边上有大块的平面的石板，应该是钓鱼佬搭建的，正好可以坐在上面，放空双脚。

放上自己喜欢的音乐，浪潮拍打河岸是自然的背景音乐；背包用来做靠背，江边风很大，不会很热，是一个舒适的点，经过一路的奔波正好在这里放松。

之前和yuki聊天，咱和他说要一起喝茶，一直没和yuki讲，sky不擅长这个，不太好意思呢。直到临近出发时间的聊天，尝试找yuki一同喝茶！也为这次来广东添加了一份更明亮的色彩。

难得的与推友见面，和与yuki在一起很轻松愉悦。和我相差无几，yuki有时也喜欢做公交晃啊晃，晃到要去的地方，所以咱们就从刚才的河边坐公交、走路、再做地铁抵达了广州。

鉴于不知道吃什么，下了地铁一转弯就望见了一家麦当劳，麦门！！于是就有了很甜的一次麦当劳，太甜了啊(≧∀≦)ゞ享受。

晚上终于达成了目的，一起喝咖啡吧！我一如既往的选择美式，yuki要了红茶拿铁。店门上明明有写支持ApplePay，应该是店员不太懂系统，点了半天屏幕说不行(￣▽￣)"

一起喝点茶or咖啡or酒，在每个不一样的地方，都可以带来不一样的体验，它们可以包容我的各种喜怒哀乐，给我带来许多。故事渐渐的开始向[《青春咖啡馆》](https://book.douban.com/subject/4714018/){target="_blank"}中所出现的那样，字字句句都映射出来，渐渐的让我怀念与感慨。

夜幕降临就是广州灯红酒绿的开始，灯火璀璨，繁华喧嚣，甚至有人在岸边放烟花。咱们上了洛溪大桥，开始返程，这上面可以望见广州塔、摩天轮，还有各种大厦，与白天相比，简直是两样。只是行人道上车子好多。

下桥，出发向地铁站，转向高铁站，附上礼物送yuki回去，当走向反方向是，心中放映了一秒那个下午，我好幸福

## ***再见***

好像什么也没去么，哈哈，根本没有计划，是这样的，甚至当时酒店都是送走yuki后订的，那时已经快11点了。咱也不喜欢那种打卡式、特种兵式，还有去那种人很多的景点玩儿。

在陌生的城市漫步，没有目的地，或者仅有寥寥数及的方向，留下大把的空闲时间给自己。走着，走着，走着，来过这里，有看到了什么，咱都感觉达到了咱的目的!

或许到时候我也会怀恋这里的什么，然后来这里待上几年也说不定<(￣︶￣)↗[GO!]
